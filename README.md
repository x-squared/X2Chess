# X2Chess

Runnable chess PGN viewer baseline.

Current scope:
- one chessboard view
- one PGN input area below the board
- navigation controls to replay parsed moves on the board

## Targets

The project is configured as fat-client-first:

- Full browser fat-client (primary profile, local execution)
- Reduced app profile (Tauri wrapper over the same frontend codebase)
- Browser client may be launched from remote hosting, but execution remains local.

## Environment setup

Use the setup manual: `doc/setup-manual.qmd`

## Run (web)

- Frontend: `cd frontend && npm install && npm run dev`
- Local game/config resources are handled directly in the client via local folder selection.

## Run (desktop)

- Dev mode (hot-reload web UI inside desktop shell):
  - `cd frontend && npm install && npm run desktop:dev`
- Package standalone app assets:
  - `cd frontend && npm run desktop:pack`
  - packaged output is generated by Tauri in `frontend/src-tauri/target/release/bundle`
