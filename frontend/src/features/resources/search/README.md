# `features/resources/search`

Search panels for the resource feature.

Important files:

- `GameSearchPanel.tsx`
- `PositionSearchPanel.tsx`
- `TextSearchPanel.tsx`

These panels are rendered from the shell’s right panel stack and consume resource search infrastructure from the top-level `resources/` package and feature hooks.

Resource search panels.
